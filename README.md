# Console API Caller App
### By Rochelle Roberts
-----

## Description
* Simple console app used to make API calls

## Technologies Used
* C#/.NET
* MySQL
* Entity Framework

## Installation
* Follow the following stepts using the terminal

```sh
$ git clone https://github.com/rochellev/ConsoleApiCall.git
$ cd \ConsoleApiCall
$ dotnet restore
$ dotnet run
```
